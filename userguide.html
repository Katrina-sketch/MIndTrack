User Guide

Introduction

This user manual provides instructions on how to use MindTrack, a mobile app designed to help users manage mental health, track habits, and stay productive.

Technical Specifications

  • Internet required for syncing and data backup
  • Optimized for smartphone screens (5"–7")
  • Platform: Android / iOS (Prototype stage)
  • System Requirements: Android 9+ / iOS 12+, RAM 2GB, Storage 100MB free
